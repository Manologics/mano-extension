## What It Does
The Mano Chrome Extension is the mobile command interface for the Monkee Biz AI revenue automation system. It connects directly to the Mano SAOS backend, giving the operator real-time access to leads, bookings, alerts, and AI-powered page analysis — from any tab, on any page.

## Features
- 🔴 Hot lead monitoring — live pipeline view
- 📅 Booking tracker
- 🚨 Ops alerts feed
- ⚡ Daily report pull
- 🤖 Claude-powered page analysis (Anthropic API)
- 💬 Direct Mano AI chat interface

## Tech Stack
- Manifest V3 Chrome Extension
- Claude (Anthropic) for page reading + AI analysis
- Base44 API for live pipeline data
- Manologics backend via mano-core-functions

## Setup
1. Clone this repo
2. Go to `chrome://extensions` → Enable Developer Mode
3. Click Load Unpacked → select this folder
4. Open Settings tab → add your Base44 + Anthropic API keys
5. You're live 🟢
